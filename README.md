# my-project

## 拡張カルマンフィルタと制御シミュレーション仕様

### 概要
このプロジェクトでは、拡張カルマンフィルタ (EKF) を用いてセンサデータ（傾き、加速度）から自己位置推定を行い、PD制御による速度制御とxy軸の制振制御を組み合わせたシミュレーションを実装します。自己位置推定しながら自己姿勢を監視する機能を追加します。

### システムモデル
- **対象**: 2D平面上のロボット（例: 地上ロボットやドローンの水平運動）。
- **状態ベクトル**: [x, y, vx, vy, θ, ω]
  - x, y: 位置
  - vx, vy: 速度
  - θ: 姿勢（ヨー角）
  - ω: 角速度
- **運動モデル**: 定加速度モデルを仮定。
  - 予測: 位置 = 位置 + 速度 * dt + 0.5 * 加速度 * dt^2
  - 速度 = 速度 + 加速度 * dt
  - 姿勢 = 姿勢 + 角速度 * dt

### センサモデル
- **データソース**: sensor_jog.txt の測定データを使用。
- **測定値**:
  - CompX(deg), CompY(deg): 傾き角度（ロール、ピッチ）。
  - Vst_y(m/s), Vsy(m/s), Vsx(m/s): 速度データ。
- **ノイズ**: 実際のデータにノイズを追加（σ_a=0.1, σ_ω=0.05）。
- **測定モデル**: EKFの更新ステップで使用。CompX, CompYを姿勢推定に、速度を位置推定に活用。

### 拡張カルマンフィルタ (EKF)
- **予測ステップ**:
  - 状態遷移: F * x + B * u (u: 制御入力)
  - 共分散: P = F * P * F^T + Q
- **更新ステップ**:
  - 測定残差: z - h(x)
  - カルマンゲイン: K = P * H^T * (H * P * H^T + R)^-1
  - 状態更新: x = x + K * (z - h(x))
  - 共分散更新: P = (I - K * H) * P

### PD制御
- **速度制御**: 目標速度 (vx_des, vy_des) に対するPD制御。
  - 誤差: e_vx = vx_des - vx_est, e_vy = vy_des - vy_est
  - 制御出力: u_x = Kp * e_vx + Kd * (e_vx - e_vx_prev)/dt
  - 同様に u_y
- **パラメータ**: Kp=1.0, Kd=0.1 (調整可能)

### xy軸の制振制御
- **振動抑制**: 位置と速度のフィードバックでダンパー効果。
  - 制御出力: u_damp_x = -K_damp * vx_est - K_pos * (x_est - x_ref)
  - 同様に u_damp_y
- **パラメータ**: K_damp=0.5, K_pos=0.2 (調整可能)

### 自己姿勢監視
- **監視機能**: EKF推定姿勢 θ_est をリアルタイム監視。
  - 閾値: |θ_est| > θ_threshold で警告。
  - ログ出力: 姿勢の履歴と異常検知。

### シミュレーション設定
- **言語**: Python
- **ライブラリ**: NumPy, SciPy, Matplotlib, Pandas (データ読み込み用)
- **時間ステップ**: dt = 0.01 s
- **シミュレーション時間**: sensor_jog.txt のデータ長に基づく（約10 s）
- **初期条件**: x=0, y=0, vx=0, vy=0, θ=0, ω=0
- **ノイズ**: σ_a=0.1, σ_ω=0.05
- **プロセスノイズ**: Q 対角行列
- **測定ノイズ**: R 対角行列
- **データ読み込み**: sensor_jog.txt をPandasで読み込み、測定値として使用。
